## プログラミング例

### Python
```python
def greet(name):
    """シンプルな挨拶関数"""
    return f"Hello, {name}!"

# 関数を呼び出す
print(greet("World"))
```

### JavaScript
```javascript
// ES6のアロー関数を使用
const greet = (name) => {
  return `Hello, ${name}!`;
};

// 関数を呼び出す
console.log(greet('World'));
```

### Ruby
```ruby
def greet(name)
  "Hello, #{name}!"
end

# 関数を呼び出す
puts greet('World')
```
